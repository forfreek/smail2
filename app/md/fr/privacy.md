## Politique de confidentialité

_Dernière mise à jour : 2026-02-12_

### 1. Vue d'ensemble

Cette Politique de confidentialité explique quelles données trashmail.space peut traiter lorsque vous utilisez le service et quelles limites s'appliquent à une boîte temporaire.

En utilisant trashmail.space, vous acceptez cette politique.

### 2. Quelles informations collectons-nous ?

trashmail.space est conçu pour limiter la collecte aux données nécessaires au fonctionnement du service.

- **Contenu des emails** : Les messages envoyés vers des adresses temporaires peuvent être conservés jusqu'à 24 heures afin d'être affichés dans la boîte.
- **Données techniques** : Des informations techniques de base comme l'adresse IP, le type de navigateur, l'horodatage et des journaux de requêtes peuvent être traitées pour l'exploitation et la sécurité.
- **Cookies ou stockage local** : Le service peut utiliser un stockage local minimal pour les préférences ou le bon fonctionnement de l'interface.

### 3. Pourquoi ces données sont-elles traitées ?

Ces données peuvent être utilisées pour :

- faire fonctionner le service de réception temporaire
- afficher les messages dans la boîte
- limiter les abus, le spam, les fraudes et les usages malveillants
- diagnostiquer des problèmes techniques et améliorer la stabilité

### 4. Durée de conservation

Les messages temporaires peuvent être conservés jusqu'à 24 heures puis supprimés automatiquement.

Les journaux techniques peuvent être conservés plus longtemps lorsque cela est nécessaire pour la sécurité, la prévention des abus ou la maintenance.

### 5. Vos responsabilités

N'utilisez pas trashmail.space pour :

- des données personnelles sensibles
- la banque, le juridique ou le travail
- des comptes dont la récupération à long terme est importante

Une boîte temporaire n'est pas conçue pour le stockage durable ni pour les usages à haute confiance.

### 6. Services tiers

Le service peut s'appuyer sur une infrastructure, des outils d'analyse ou des composants tiers pour fonctionner. Ces tiers peuvent traiter certaines données techniques selon leurs propres politiques.

### 7. Modifications de cette politique

Cette politique peut être mise à jour à tout moment. La date de mise à jour sera modifiée en haut de la page lorsque des changements importants seront apportés.

### 8. Contact

Si vous avez des questions sur cette politique, utilisez les canaux de contact indiqués sur le site.
